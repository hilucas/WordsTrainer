# Word Trainer Pro 单词训练专家

这是一个专为英语学习者设计的高级背词工具，结合了科学的 SM2 记忆算法和现代化的玻璃拟态（Glassmorphism）设计语言。无需服务器，通过浏览器本地存储实现学习进度的管理。

## ✨ 核心功能

- **🚀 SM2 记忆算法**：基于科学的间隔重复算法，自动安排复习时间，并可视化显示每一个单词的掌握度和下次复习间隔。
- **🎨 Premium 视觉设计**：极具质感的深色玻璃拟态 UI，支持流畅的动效反馈，专为人格力学优化的移动端底部操作手势。
- **📚 300+ 高频词库**：预置 300 多个涵盖 8 类核心主题的高频词汇（情感、行为、特征、学术、短语等）。
- **📊 可视化统计**：支持环形进度条和宫格数据卡片，实时反馈词库总量、已学单词、掌握程度及学习进度。
- **🔊 语音朗读**：支持 Web Speech API 实现在线单词朗读，加速发音记忆。
- **💾 零成本数据同步**：无服务器备份方案，支持通过剪贴板一键导出/恢复进度，方便在不同设备间迁移数据。

## 📖 词库概况

本项目目前包含 **300+ 个核心单词/短语**，并划分为以下 8 大核心领域：
1. **态度 / 情感** (Attitude & Emotion)
2. **行为 / 动作** (Action & Behavior)
3. **特征 / 属性** (Property & Character)
4. **数量 / 范围** (Quantity & Scope)
5. **状态 / 趋势** (State & Trend)
6. **学术 / 专业** (Academic & Professional)
7. **固定短语 / 习语** (Idioms & Phrases)
8. **派生 / 变形词** (Derived Words)

## 📥 导入与导出指南

### 进度备份 (Export)
1. 进入“导入导出”页面。
2. 点击“导出备份”按钮，学习进度和收藏列表将自动复制到您的剪贴板。
3. 将该文本保存在文档或笔记中即可。

### 进度还原 (Import)
1. 进入“导入导出”页面。
2. 点击“恢复备份”按钮。
3. 在弹出的窗口中粘贴之前导出的备份文本。

### 批量词汇导入 (Bulk Word Import)
- **格式要求**：`单词, 读音, 含义, 分类` (CSV 风格)
- **示例**：`apple, /ˈæpl/, 苹果, fruit`
- 在导入页面的文本框中按行粘贴后点击“导入词表”即可立即生效。

## 🛠️ 技术栈
- **Frontend**: Vanilla JavaScript (ES6+), CSS3 (Custom Properties), HTML5.
- **Algorithm**: SM2 (Spaced Repetition).
- **Storage**: Browser LocalStorage.
- **PWA**: 支持离线缓存 (Service Worker)。

## 🚀 如何运行
本程序为纯静态网页，您可以直接下载源码并点击 `index.html` 在浏览器中运行，或部署在 GitHub Pages 等平台上。
